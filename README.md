# Portfolio de Juan Camilo Pájaro Reyes

Portafolio profesional de desarrollador fullstack construido con Next.js, React y TypeScript.

## Características

- Diseño responsive y moderno
- Secciones: Hero, Acerca de mí, Proceso, Habilidades, Proyectos y Contacto
- Animaciones interactivas con hover effects
- Iconos de tecnologías que aparecen al pasar el mouse
- Formulario de contacto funcional

## Tecnologías Utilizadas

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Lucide Icons

## Instalación

\`\`\`bash
# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm run dev

# Construir para producción
npm run build

# Ejecutar pruebas
npm test

# Ejecutar pruebas en modo watch
npm test:watch

# Generar reporte de cobertura
npm test:coverage
\`\`\`

## Pruebas Unitarias

El proyecto incluye pruebas unitarias con Jest y React Testing Library:

- Tests de componentes principales (Header, Hero, Skills, Projects, Contact)
- Tests de utilidades
- Configuración completa con coverage

Para ejecutar las pruebas:
\`\`\`bash
npm test
\`\`\`
